# Bitmaps to Images

- bitmaps are saved in the bitmaps directory of the project
- bitmaps were created with GIMP using ChatGPT instructions on setup
- use GIMP to edit BMP's (see below)
- use <https://javl.github.io/image2cpp/> to translate to C code
- set invert image colors, bitmap should be mostly 0's and 1's where we color
- set swap bits in byte
- place code in include/bmp.h
